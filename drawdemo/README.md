# Keras MNIST Demo

This project demonstrates MNIST digit classification using Keras.

## Setup

1. Create a virtual environment:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows: .venv\Scripts\activate
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Open `mnist_demo.ipynb` to explore the dataset and train the model.
